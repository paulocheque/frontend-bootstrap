App Web
============

Getting Started
----------------

    # Install global tools
    fab npm

    # Bootstrap the project
    npm install
    bower install

    # Test it locally
    fab localhost server
    # or
    fab dev_server
    # or
    fab prod_server
    # or
    gulp serve
    # or
    gulp serve:dist
